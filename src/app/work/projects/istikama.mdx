## Overview

Istikama is a highly customized and modernized Moodle-based educational ecosystem designed to streamline school management, content delivery, and technical moderation. The platform facilitates a robust workflow for teachers, students, and technical professors across various academic levels. 

The primary goal of this project was to transform a standard Moodle setup into an intuitive, dynamic, and modern educational application that handles everything from interactive content generation to dynamic status tracking.

## Key Features

- **Advanced Content Bank Moderation**: A dynamic, AJAX-driven moderation and tagging system that allows technical professors to oversee educational material intuitively without reloading the page.
- **Academic Hierarchies**: Sophisticated handling of global question banks categorized by level and subject, ensuring precise permission controls across multiple schools within the platform.
- **Multilingual Support**: Fully localized in French and Arabic, giving both instructors and learners a native experience when managing or taking courses.
- **Interactive Quizzes**: Modernized Moodle quiz interfaces seamlessly integrated with the platform's overarching theme to avoid interface breaks and enhance user engagement.
- **Real-Time Data Dashboard**: Administrative analytics and reporting to track student progress, content moderation status, and overall school performance.

## Technologies Used

- **PHP & Moodle Engine**: The core architecture powering the educational content and user management.
- **JavaScript & AJAX**: For dynamic state machines and modern, app-like interactions without page reloads.
- **Docker**: Containerized environment for smooth local development and deployment mapping to cloud infrastructure.
- **MySQL**: Optimized database schema handling high volumes of quiz data, user logs, and course content.
